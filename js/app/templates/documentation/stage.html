Backbone.Stage

In a Backbone+ application there is one special extended scene that represents the stage. The stage will help with the task of managing your application's viewport properties and current scene.

Creating a Backbone.Stage

Most likely you will want to create your stage after DOM ready has fired. Also, you will want to attach a new router instance to the stage. This way the stage can help with scene changes.

// App Ready
$(document).ready(function() {
	App = new Backbone.Stage();
	App.router = new AppRouter();
});
