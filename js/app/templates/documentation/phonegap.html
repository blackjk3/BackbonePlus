Backbone.Phonegap

Optional helper class for common Phonegap operations.

Functions

pause() & resume()

Internal methods that will fire if your app goes into the background or resumes from the background. Backbone+ will automatically shut down any timers / animations when the app goes into the background.

connection()

Checks the active network connection that is being used. Returns the current connection.

	var conn = Backbone.PhoneGap.connection();
	// conn => Connection.CELL_3G
