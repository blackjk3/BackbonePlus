Backbone.Element

A Backbone.Element is the fundamental building block of all visual objects within Backbone+. It inherits from Backbone.View and is extendable. Properties can be set using accessor methods and will be redrawn using CSS3 transforms when possible for HW acceleration.

Creating an Element

var element = new Backbone.Element({ tagName: 'footer' });

Extending an Element

var NewAndImprovedElement = Backbone.Element.extend({
    initialize: function(options) {},
    render: function() {},
    newFunctionThatRocks: function() {}
});

var element = new NewAndImprovedElement();

Functions

set( name, value )

Sets a specific property on a Backbone.Element. Set will trigger a redraw. Redrawn properties include x, y, z, w, h, alpha, rotation, and scale.

element.set('x', 200);
attr( args )

Sets multiple properties on a Backbone.Element. Attr also triggers a redraw.

element.attr({x:400, y:300, z:1, w:200, h:200, rotation:45, alpha:0.3}});
css( args )

Sets multiple css properties at once.

element.css({borderRadius: '1px', color: '#000'})
tween( args )

Animates a Backbone.Element over a specified duration. Under the hood, tween will use a requestAnimationFrame or setInterval to calculate each frame and preform a redraw accordingly. Tween is great for animations that are time dependent or require regular progress updates.

Arguments: to: object, duration: milliseconds, delay: milliseconds, easing: see (Backbone.Easing), complete: callbackFunction, progress: callbackFunction (sends a pct complete)

var tween = element.tween({
    to: { scale: 0, rotation: 100 },
    duration: 10000,
    complete: completeCallback, 
    progress: progressCallback,
    easing: Backbone.Easing.Bounce.EaseOut
});
removeTween( tween )

Stops and removes a tween from a Backbone.Element.

element.removeTween(tween);
removeAllTweens()

Stops and removes all tweens attached to a Backbone.Element.

element.removeAllTweens();
delay( args )

Sets up a generic delay timer. Useful for all sorts of internal timing events within an app or game.

element.delay({
    duration: 1000,
    complete: completeCallback, 
    progress: progressCallback
});
